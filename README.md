# 💻 Teste de Desenvolvimento – Exercícios em C#

Este repositório reúne uma coleção de exercícios desenvolvidos como parte de um **teste técnico de desenvolvimento**, com o objetivo de demonstrar conhecimentos em **lógica de programação, resolução de problemas e conceitos fundamentais utilizando C#**.

Os exercícios foram organizados em pastas individuais, cada uma contendo a implementação de um problema específico. A proposta foi trabalhar diferentes níveis de complexidade, desde operações básicas até conceitos mais estruturados como **manipulação de coleções e programação orientada a objetos**.

# 🧠 Tecnologias Utilizadas

* C#
* .NET
* Aplicações Console
* Visual Studio
